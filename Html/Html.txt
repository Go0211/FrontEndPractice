<Body>
1. Top / Left Margin="수치"	: 여백지정
2. BackGround="주소" : 배경이미지
3. BgColor="색상" : 배경 색상
4. Text="색상" : 글자색 
5. null / A / V Link="링크" : 링크색상 / 마우스 클릭시 색상 / 이미 방문한 색상

	<Font> : 글자
	1. Size="수치" : 글자크기
	2. Color="색상" : 글자색상
	3. Face="글꼴" : 글꼴 종류

	<UL> : 기호 목록
	● <Li> type="모양" : 모양지정

	<OL> : 숫자 목록
	1. type="숫자형식" : 숫자형식
	2. start="숫자" : 시작 숫자
	● <Li> type="숫자형식" : 숫자형식
	           value="숫자" : 항목번호를 중간에 바꿀 경우
		
	<Hr> : 수평선
	1. Color="색상" : 색상 지정
	2. Width="수치" : 가로 크기
	3. Size="수치" : 세로 크기
	4. NoShade : 입체 해제
	5. Aligh="정렬" : 정렬 지정

	<Img> : 이미지
	1. Src="주소" 	: ※필수, 주소지정
	2. Width="수치"	: 가로 크기
	3. Height="수치"	: 세로 크기
	4. Alt="내용"	: 설명 지정
	5. Border="수치"	: 테두리 두께 
	6. VSpace="수치"	: 그림 가로 여백
	7. HSpace="수치"	: 그림 세로 여백
	8. Align="정렬"	: 정렬 방법
	9. UseMap="#이름"	: 그림 맵의 지정

	※자세히
	<A> : 글자나 그림 or 웹 페이지를 연결
	1. Href="주소"	: ※필수, 링크 주소 (특정 페이지는 #~~)
	2. Name="이름"	: 문서의 특정 위치 
	3. Target="속성"	: 링크가 열릴 프레임 방식 지정 
	4. title="내용"	: 링크에 마우스 올렸을 때 보이는 내용

	<Table> : 표
	1. Width="수치" 		: 너비
	2. Height="수치" 		: 높이
	3. Border="수치"		: 테두리 두께 지정 (안하면 없음)
	4. Align="정렬"		: 정렬
	5. CellSpacing="수치 "	: 각 셀 간의 간격 지정
	6. CellPadding="수치"	: 산 안의 내용 간의 간격 지정

		<Tr> : 행
		1. Align="정렬"	: 수평 정렬
		2. VAlign="정렬"	: 수직 정렬
		3. BGColor="색상"	: 셀의 배경색 지정

			<Td> : 열		(제목으로 하려면 <Th>를 사용)
			1. Width="수치" 		: 너비
			2. Height="수치" 		: 높이
			3. Align="정렬"		: 수평 정렬
			4. VAlign="정렬"		: 수직 정렬
			5. RowSpan="수치"		: 세로로 셀을 합침
			6. ColSpan="수치"		: 가로로 셀을 합침
			7. BGColor="색상"		: 셀의 배경색 지정

	<Input> : 텍스트 박스, 버튼
	1. Type="타입"		: 입력 양식의 타입 지정 (text, radio, checkbox, reset등)
	2. Name="이름"		: 값에 대한 이름 지정
	3. Value="값"		: 양식의 기본 값
	4. Size="수치"		: 양식의 크기
	5. MaxLength="수치"	: 문자의 최대값
	6. Checked		: ('checkbox', 'radio') 초기값으로 선택할 것 지정

	※ type="submit" : 전송버튼	/ reset : 취소버튼
	   value="다른이름" : 다른이름

	※ <Form> 
	    Action="Url" 		: 값을 전송할 주소 지정
	    Method="전송방식" 	: 전송방식 지정	

	<TextArea> : 여러줄의 텍스트 박스
	1. Rows="수치"	: 세로 줄 수 지정
	2. Cols="수치"	: 가로 문자 수 지정 
	3. Name="이름"	: 이름
	4. ReadOnly	: 읽기만 가능

	<Select> : 콤보박스 틀
	1. Size="수치"		: 화면에 보이는 항목 수
	2. Name="이름"		: 이름
	3. Multiple		: 여러개 선택 가능

		<Option> : 콤보박스의 요소들
		1. Value="값"	: 항목의 값 지정
		2. Selected	: 기본 값
		3. Disabled 	: 항목 비활성화
